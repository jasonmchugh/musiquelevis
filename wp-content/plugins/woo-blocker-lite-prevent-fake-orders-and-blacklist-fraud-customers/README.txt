== Frequently asked questions ==

= Q. How do I block fraud customers on checkout page? =

Yes, you have to activate woocommerce blocker plugin and enable checkout page condition in the plugin and you can block specific user by email, IP address, state and zipcode

= Q. How do I block  fraud customers on registration pages? =

Yes, you have to activate woocommerce blocker plugin and enable registration page condition in the plugin  and you can block specific user by email, IP address, state and zipcode

= Q. Can I block fraud customers by IP address? =

Yes, you have to activate woocommerce blocker plugin and enable registration page/ checkout page condition in the plugin  and you can block IP address. You can add multiple IP address in the plugin.

= Q. Can i block fraud customers by state, zip code, domain, IP address and email address? =

Yes, you have to activate woocommerce blocker plugin and enable registration page/ checkout page condition in the plugin  and you can block IP address. You can add multiple IP address in the plugin.

= Q. Can i set custom error message for blocked listed user? =

Yes, you have to activate woocommerce blocker plugin and enable registration page/ checkout page condition in the plugin  and you can block IP address. You can add multiple IP address in the plugin. Plugin will give option to add custom message as per your condition like block by email, IP address, state and zipcode.

= Q. Can I block external blacklist of github with woocommerce blocker plugin? =

Yes, you have to activate woocommerce blocker plugin and enable <a href="https://github.com/kowsar89/temporary-email-domains/">external blacklist of github</a>. There is a list of disposable/temporary email domains in Github. You can use this list too along with your own blacklist.

= Q. Can I see list of blocked users/ top blocked user and how many times he has attempted for placing an order in the website. =

Yes, you have to activate woocommerce blocker plugin. Plugin provides list of blocked users email address and how many attempt by specific user.
